Week 4 Tech Blog: Enumerable#cycles

I just learned about the #cycle method and I feel like every budding Rubyist should know about it.

Cycle is an iterator similar to each except within an infinite loop. You can always use #times to iterate a specific number of times. Sometimes you want to loop through an array an arbitrary or infinite number of times. I haven't disovered a reason to do this, but I'm super green when it comes to Ruby. Take a look at this example.

Forever and ever. Try passing it an argument.

Passing an argument to the cycle will loop however many times you specify. If you don't, it will loop infinitely. Pretty cool.
